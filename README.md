## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

Reto - Backend

El reto consiste en realizar una pequeña API para calcular el tipo de cambio, el
endpoint de la API debe recibir el valor “monto“, “moneda origen”, “moneda destino“
y devolver el “monto”, “monto con tipo de cambio”, “moneda origen”, “moneda
destino“ y “tipo de cambio”.
Algunas consideraciones para el desarrollo del reto:
● El lenguaje de programación a utilizar es Node JS con NestJS.
● Implementar un nivel de seguridad en la consulta (JWT).
● Se debe crear la información del tipo de cambio en una in memory database
● Dockerizar la solución e invocar a la API desde el contenedor
● El uso de la API debe ser mostrada desde Postman.
● Realizar pruebas unitarias
Si tuvieras alguna duda en el momento de realizar el caso, te recomendamos que
tomes una decisión en base a tus conocimientos y experiencia, y en el momento de
presentar tu desarrollo sustentarías el por qué de esta decisión.
También te dejamos algunas funcionalidades opcionales que puedes incorporar en
la solución del caso:
● Documentar el API con Swagger u Open api
● Crear un POST para actualizar el valor del tipo de cambio.
● Desplegar el API en AWS. (Considerar la infraestructura que mejor se te
acomode)
● Utilizar NestJs con Fastify
● Subir el código a cualquier repositorio de código fuente
● Implementar control de excepciones
Cuentas con 3 días para desarrollar el reto, pasado ese tiempo debes enviarnos el
link del repositorio de código fuente.
